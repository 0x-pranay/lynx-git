# GIT


## Install git
  sudo apt install git-all


## Configure 
  git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com


## Initialize GIt

git init

## Commit

 - Add changes
 - add the file to stating
   git add <file_name>

 - commit the change
    git commit 

     (or)

   git commit -m 'Commit message'
